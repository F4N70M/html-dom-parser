# Примеры использования HtmlDomParser

В этой папке находятся примеры, демонстрирующие различные возможности библиотеки.

## Требования

- Установленная библиотека через Composer (см. корневой README)
- PHP 7.4+

## Запуск примеров

```bash
php examples/01-basic-usage.php
php examples/02-working-with-fragments.php
php examples/03-error-handling.php
```

**Важно:** В текущей версии библиотеки метод `Parser::parse()` ещё не реализован, поэтому примеры не будут выполняться. Они служат иллюстрацией ожидаемого API и поведения.

## Содержание

- `01-basic-usage.php` – базовый парсинг, получение элементов, атрибутов, data/label.
- `02-working-with-fragments.php` – работа с фрагментами форматированного текста после схлопывания строчных элементов.
- `03-error-handling.php` – настройка обработчика ошибок, сбор ошибок, узлы-ошибки в дереве.

Подробная документация доступна в [основном руководстве](../docs/README.md).
