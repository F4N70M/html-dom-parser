# Установка и начало работы {#installation}

В этом разделе описаны системные требования и процесс установки библиотеки HtmlDomParser через Composer.

## Предварительные требования {#prerequisites}

Перед установкой убедитесь, что ваше окружение соответствует следующим требованиям:

- **PHP версии 7.4+** (рекомендуется 8.0 или выше)
- **Расширение PHP `ext-dom`** — обычно включено по умолчанию в стандартных сборках PHP
- **Composer** — менеджер зависимостей для PHP

> **Важно:** Библиотека использует `DOMDocument` для загрузки HTML, поэтому расширение `ext-dom` обязательно. Без него парсинг будет невозможен.

## Установка через Composer {#installation-via-composer}

Выполните в терминале следующую команду:

```bash
composer require f4n70m/html-dom-parser
```

В процессе установки Composer:
1. Загрузит библиотеку и её зависимости
2. Зарегистрирует автозагрузку классов
3. Создаст или обновит файл `composer.json` и `composer.lock`

## Проверка установки {#verifying-installation}

Создайте файл `test.php` и добавьте следующий код для проверки:

```php
<?php

require_once __DIR__ . '/vendor/autoload.php';

use HtmlDomParser\Parser;

$parser = new Parser('<div>Hello, World!</div>');
$document = $parser->parse();

echo 'Установка прошла успешно!' . PHP_EOL;
echo 'Содержимое: ' . $document->getChildren()->get(0)->getLabel() . PHP_EOL;
```

Запустите скрипт:

```bash
php test.php
```

Ожидаемый вывод:
```
Установка прошла успешно!
Содержимое: Hello, World!
```
